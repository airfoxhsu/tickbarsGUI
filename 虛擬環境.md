# 用py建立虛擬環境
1. 列出安裝的所有Python版本，有「*」星號的是預設版本:
    ```cmd 
    py -0
    -3.10-64 *
    -3.10-32
    -3.9-32
    ```
2. 自帶的虛擬環境venv
   py -3.12(python版本) -m venv myvenv(虛擬環境名稱)  
   或者
   python -m venv myvenv
   python指令只能執行它對應的那個 Python 版本（通常是你安裝的最後一個 Python），不像 py 那樣可以選擇不同版本。

 
3. 檢查目前環境：
   ```cmd
   py -m site
   ```

4. 進入虛擬環境:
   轉換到虛擬環境
   要使用剛剛建立的虛擬環境, 必須啟用 (activate) 它, 在 Windows 上必須執行虛擬環境路徑的 Scripts 資料夾下的 activate.ps1 指令(Powershell)：
   使用命令提示字元, 要執行的是 activate.bat

5. 退出虛擬環境
   - deactivate 離開虛擬環境
   - exit 離開虛擬環境
   -  刪除虛擬環境
      若要刪除虛擬環境, 由於虛擬環境的資料的資料全部都在單一資料夾中, 所以只要先離開虛擬環境, 再將虛擬環境的資料夾刪除即可。
  
6. 如果沒有使用虛擬環境，而是直接在全局環境中切換 Python 版本，則系統將使用與你最後一次安裝的 Python 版本相關聯的 pip。
    
7. 更新依附的 Python 版本
   如果你安裝了其他版本的 Python, 想要讓虛擬環境改用特定的 Python, 只要使用 --upgrade 選項更新虛擬環境即可。例如以下原本建立虛擬環境的時候, 使用的是 Python 3.10.9：
   ```cmd
   > .\testenv\Scripts\activate
   (testenv) > python --version
   Python 3.10.9
   (testenv) > deactivate
   ```
   而現在系統上預設的 Python 是 3.11.2 版：
   ```
   > python --version
   Python 3.11.2
   ```
   透過以下指令, 即可讓現有的虛擬環境改用 3.11.2 版：
   ```
   > python -m venv --upgrade testenv
   > .\testenv\Scripts\activate
   (testenv) > python --version
   Python 3.11.2
   (testenv) >
   ```
   這並不限於更新版本, 也可以轉換到舊到的版本。

8. 使用 py 啟動器來更改預設版本（推薦）
   Windows 預設的 Python Launcher (py) 會根據設定來選擇 Python 版本，你可以更改 py.ini 配置文件來指定預設版本。
   步驟
   找到 py.ini 配置文件
   推薦：
       在C:\Windows\目錄下新建一個名為的文件，其中寫入如下內容，就可以修改python 的預設版本：py.ini
   不推薦：    
       py 會讀取 %LOCALAPPDATA%\Programs\Python\Launcher\py.ini（有時可能不存在，需要自己建立）。
   你可以在 cmd 中輸入：   
   notepad %LOCALAPPDATA%\Programs\Python\Launcher\py.ini
   
   如果這個文件不存在，則需要手動創建。
   設定預設版本 在 py.ini 文件中添加：
   [defaults]
   python=3.9
   這樣 py 就會預設使用 Python 3.9，而不是系統中最新安裝的版本。

   測試
   py  # 這應該會啟動 Python 3.9
   py --version  # 檢查是否成功

# How do I install a Python package with a .whl file?
pip install C:\some-dir\some-file.whl
pipenv install C:\some-dir\some-file.whl

ex: pipenv install C:\Users\b2465\.virtualenvs\Zero_judge-aaF_ihw8\Scripts\haohaninfo-0.2.23-cp39-none-win32.whl

# 安裝特定版本模組:
py -m pip install "SomeProject==1.4"
To install greater than or equal to one version and less than another:
py -m pip install "SomeProject>=1,<2"
To install a version that’s compatible with a certain version:
py -m pip install "SomeProject~=1.4.2"
In this case, this means to install any version “==1.4.*” version that’s also “>=1.4.2”.
